# DevRoast

Paste your code. Get roasted.

DevRoast é uma aplicação web onde você cola seu código e recebe uma análise automática com feedback bem humorado sobre a qualidade do código. Perfeito para quem quer se diverir com as falhas do próprio código ou simplesmente quer uma opinião "honesta" sobre seu código.

## Funcionalidades

- **Análise de Código** - Cola seu código em várias linguagens e recebe uma nota de 0 a 10
- **Feedback Divertido** - Receba "roasts" bem humorados sobre os problemas encontrados
- **Sugestões de Melhoria** - Veja sugestões de como melhorar seu código
- **Leaderboard** - Compare sua pontuação com outros desenvolvedores
- **Modo Escuro** - Suporte a tema claro e escuro
- **Compartilhamento** - Compartilhe seus resultados

## Como Usar

1. Acesse a aplicação
2. Cole seu código na área de texto
3. Clique em "roast my code"
4. Receba sua pontuação e feedback!

```bash

# Run development server
npm run dev
```
